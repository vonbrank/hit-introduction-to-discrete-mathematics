1.1.1. 8.

令 TTT 是一个正则二元树，它有 iii 个内顶点（出度为 222 的顶点）。如果 EEE 为所有内顶点深度之和， III 为所有叶节点的深度之和，证明：

I=E+2iI = E + 2iI=E+2i

证明：

设 TTT 有 ppp 个顶点，施归纳于 ppp :

  • p=1p = 1p=1 时， I=E=i=0I = E = i = 0I=E=i=0 。
  • p≠2p \neq 2p=2 。
  • p=3p = 3p=3 时， I=2I = 2I=2 ， E=0E = 0E=0 ， i=1i = 1i=1 ，结论成立。

假设节点数小于 ppp 时成立，往证节点数为 ppp 时也成立：

若去掉根节点及其边，产生的两棵子树节点树均小于 ppp ，符合归纳假设。设这两棵子树满足：

I1=E1+2i1(1)I_1 = E_1 + 2 i_1 \tag{1}I1​=E1​+2i1​(1) I2=E2+2i2(2)I_2 = E_2 + 2 i_2 \tag{2}I2​=E2​+2i2​(2)

加回根节点后，两棵子树所有节点深度均加 111 。故：

I=I1+I2+(p−i1−i2−1)I = I_1 + I_2 + (p - i_1 - i_2 - 1)I=I1​+I2​+(p−i1​−i2​−1) E=E1+E2+i1+i2E = E_1 + E_2 + i_1 + i_2E=E1​+E2​+i1​+i2​ i=i1+i2+1i = i_1 + i_2 + 1i=i1​+i2​+1

仅证需证明 I1+I2+(p−i1−i2−1)=E1+E2+i1+i2+2(i1+i2+1)I_1 + I_2 + (p - i_1 - i_2 - 1) = E_1 + E_2 + i_1 + i_2 + 2 (i_1 + i_2 + 1)I1​+I2​+(p−i1​−i2​−1)=E1​+E2​+i1​+i2​+2(i1​+i2​+1)

由 (1)(1)(1) 式和 (2)(2)(2) 式，化简得 p=2(i1+i2+1)+1p = 2(i_1 + i_2 + 1) + 1p=2(i1​+i2​+1)+1 ，即 p=2i+1p = 2i + 1p=2i+1 ，由于 TTT 是正则二元树，此式显然成立，故结论成立。【证毕】
